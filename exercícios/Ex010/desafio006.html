Tags HTML5

Em desenvolvimento HTML5, podemos usar os elementos apresentados abaixo(Arraste o cursor do mouse por cima de cada elemento para saber qual é a tag usada para a criação de cada um):

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Codigos-fonte
  • Listas numeradas
  • Listas com demarcadores
  • Hyperlinks
  • Textos Sublinhados
  • Textos excluídos
  • Textos em itálico
  • Textos em destaque ou negrito
  • Textos marcados
